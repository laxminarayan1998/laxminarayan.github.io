I'm Laxminarayan, a 25years old self-taught Mobile Application Developer, from Bhubaneswar.

I started using a smartphone when I was in the 6th standard, then the apps in it attracted me, and was curious how these apps are being made. Since then I decided to build my career in the IT field.

In August 2017, I took admission to Computer Science & Engineering in Orissa Engineering College, thought now I would learn about the technology in college, where I competed for 2years, but I only learned basic JAVA programming language,
I wasn't learning and improving, I felt stuck.

In July 2019, We had a super cyclone hit in Odisha. Everything almost destructed. Schools, colleges had to shut down for the reconstruction. I got a chance to start learning & developing apps. So, I woke up early morning to study JAVA, Android. I quickly started to love Android. I have been studying application development full time ever since.

During this time, I took online courses like The Complete Android Oreo Developer Course - Build 23 Apps!, Android Studio Tutorial and watched countless youtube videos about android.

Luckily after some months, I got my first freelancing project where I could learn new things and apply all the knowledge that I collected from all the sources. This project indeed got me the confidence to accomplish a task within a deadline, though the app is beginner-level, but learned a lot about real-world problems & solutions. Since then I keep developing myself.

Slowly Steadily, I am shifting myself to Flutter because I found it is much easy and less code can produce good results in less time.
